## âš™ï¸ Schema Configuration

**Example:** `CONFIGS/Schema.config.json`

```json

  "table": "[YourDatabase].[dbo].[YourTable]",
  "mapping": {
    "Mfr": "MFR",

```

### ğŸ”¹ `mapping` Key Types

- `"Mfr": "MFR"` â€” Maps the `MFR` field in your JSON to the SQL column `Mfr`.
- `"value"` â€” A static value to insert.
- `"raw": true` â€” Insert the value as-is (no quotes), e.g. `GETDATE()`.
- `"raw": false` â€” Insert the value as a string (with quotes), e.g. `'JSaucelo'`.
